---
title: Deploying a Mastra Server
---

Build your application and start the generated HTTP server:

```bash showLineNumbers
@actus-ag/@mastra/cli/cli build
node .@@mastra/cli/cli/cli/output/index.mjs
```

To include telemetry for the generated server:

```bash showLineNumbers
node --import=./.@@mastra/cli/cli/cli/output/instrumentation.mjs .@@mastra/cli/cli/cli/output/index.mjs
```
