# @actus-ag/@mastra/cli/cli start

Start your built Mastra application. This command is used to run your built Mastra application in production mode.
Telemetry is enabled by default.

## Usage
After building your project with `@actus-ag/@mastra/cli/cli build` run:

```bash
@actus-ag/@mastra/cli/cli start [options]
```

## Options

| Option | Description |
|--------|-------------|
| `-d, --dir <path>` | Path to your built Mastra output directory (default: .@@mastra/cli/cli/cli/output) |
| `-nt, --no-telemetry` | Enable OpenTelemetry instrumentation for observability |

## Examples

Start the application with default settings:

```bash
@actus-ag/@mastra/cli/cli start
```

Start from a custom output directory:

```bash
@actus-ag/@mastra/cli/cli start --dir ./my-output
```

Start with telemetry disabled:

```bash
@actus-ag/@mastra/cli/cli start -nt
```
